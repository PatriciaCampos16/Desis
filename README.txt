========================================================
              INSTRUCCIONES DE INSTALACIÓN
========================================================

1. **Configuración del Entorno:**
   - Asegúrate de tener PHP instalado en tu servidor.
   - Configura y ejecuta un servidor web, utilizando XAMPP o similar.

2. **Configuración de la Base de Datos:**
   - Importa el archivo SQL proporcionado (`desis.sql`) en tu sistema de gestión de bases de datos. este archivo se encuentra en la carpeta bd que esta dentro de desis.
   - Asegúrate de configurar adecuadamente los detalles de la conexión a la base de datos en el archivo conexion.php.

3. **Archivos en la Carpeta htdocs:**
   - Coloca todos los archivos del proyecto en la carpeta principal de tu servidor web levantado localmente.

4. **Versión de PHP:**
   - Este proyecto ha sido desarrollado y probado con PHP versión 8.1.2

5. **Versión de la Base de Datos:**
   - Este proyecto ha sido desarrollado y probado con MariaDB versión 10.4.28.

6. **Acceso al Portal:**
   - Abre tu navegador y visita la URL correspondiente a tu servidor web y la carpeta del proyecto.

7. **¡Listo para Votar!**

   - Realice un formulario con bootstrap basico.
